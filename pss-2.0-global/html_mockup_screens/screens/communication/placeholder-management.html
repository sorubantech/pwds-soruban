How Placeholders Work

Dynamic Replacement Placeholders are replaced with actual data when emails are sent to recipients.
System Placeholders Built-in placeholders are managed by the system and cannot be modified or deleted.
Custom Placeholders Custom placeholders map to custom fields defined in Settings → Custom Fields.
Default Values Use {{FieldName|default:"Fallback"}} syntax to define fallback text when no data is available.
Supported Channels Placeholders work in Email templates, SMS, WhatsApp messages, and receipt templates.
